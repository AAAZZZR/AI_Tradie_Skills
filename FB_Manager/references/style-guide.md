# Post Style Guide

## Audience
English-speaking, content creator / media community.

## Tone
Conversational, confident, slightly playful. Write like a knowledgeable friend, not a corporate account. Avoid stiff or overly formal language.

## Emoji Rules
- Use 1-3 emojis per post, no more
- Place emojis at the start of a key line or end of a sentence, never mid-word
- Preferred set: 🔥 🚀 💡 🎯 ✨ 👀 📌 🎬 — rotate, don't repeat in the same post
- Never use emojis as bullet points
- No emoji in the first sentence

## Paragraph Structure
- Keep posts short: 3-5 short paragraphs max
- One idea per paragraph
- Use real line breaks between paragraphs (not \n)
- First line = hook (question, bold claim, or surprising fact)
- Last line = CTA or takeaway
- No walls of text — if a paragraph exceeds 2-3 sentences, split it

## Hashtag Strategy
- Place hashtags at the very end of the post, separated by a line break
- Use 3-5 hashtags per post
- Mix: 1-2 broad + 2-3 niche
- Broad examples: #ContentCreator #CreatorEconomy #DigitalMedia
- Niche: use topic-specific tags relevant to the post content
- Always lowercase after the #, no spaces in multi-word tags
- Never put hashtags inline with the body text

## CTA Templates
Use one per post, placed as the final line before hashtags:

- Engagement: "Drop a [emoji] if you agree."
- Discussion: "What's your take? Let us know in the comments."
- Save/Share: "Save this for later 📌"
- Link: "Full breakdown in the comments 👇"
- Action: "Try it today and tag us with your results."

## Things to Avoid
- All caps for emphasis (use sparingly if at all)
- Starting with "Hey guys" or "Hello everyone"
- Generic filler like "In today's post we will discuss..."
- More than one CTA per post
- Emojis as the entire first line
